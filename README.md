# possessive

A simple helper function that lets you get a possessive form of a string.

Courtesy of [ruby possessive gem](https://github.com/bclubb/possessive).

## Instalation

```javascript
npm install --save possessive

yarn add possessive
```

## How to use

```javascript
import possessive from 'possessive';

possessive('Chris');
// Chris’

possessive('Diane');
// Diane’s
```

Enjoy!
